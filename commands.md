## 1. Lifecycle Commands (Creation & Execution)
These commands control the birth, life, and death of processes.

| Standard Command | **Slang Command** | **Description / Output Example** |
| :--- | :--- | :--- |
| `init` | **`cook`** | **"Let him cook."**<br>Initializes a new project or configuration.<br>*Output: "Hold up, let me cook... Project recipe created."* |
| `start / run` | **`lock-in`** | **"Time to lock in."**<br>Starts the development server or main process.<br>*Output: "Locked in at localhost:3000. Don't choke."* |
| `build` | **`glow-up`** | **"The glow up is real."**<br>Compiles the code into a production bundle.<br>*Output: "Compiling... Sheeesh! Build size is looking snatch."* |
| `deploy` | **`yeet`** | **"Yeet it to the cloud."**<br>Deploys the application to production.<br>*Output: "Yeeting to prod... It landed! W in the chat."* |
| `stop / kill` | **`ghost`** | **"Ghost the process."**<br>Stops a running server or task.<br>*Output: "Server has been ghosted. Left on read."* |
| `restart` | **`spin-back`** | **"Spin the block."**<br>Restarts the service.<br>*Output: "Spinning back... We go again."* |

## 2. Operations (Movement & Fixing)
Commands for moving code and fixing problems.

| Standard Command | **Slang Command** | **Description / Output Example** |
| :--- | :--- | :--- |
| `push` | **`send-it`** | **"Just send it."**<br>Pushes code to the remote repository.<br>*Output: "Full send initiated... We ball."* |
| `pull` | **`yoink`** | **"Yoink the code."**<br>Pulls changes from remote.<br>*Output: "Yoink! Stole the latest changes from main."* |
| `fix / debug` | **`clutch`** | **"Clutch up."**<br>Runs auto-fixers or debuggers.<br>*Output: "Clutching... 1v5 situation... Fixed 3 errors. MVP."* |
| `update` | **`rizz-up`** | **"Unspoken Rizz."**<br>Updates dependencies or the CLI itself.<br>*Output: "Rizzing up packages... You got that update rizz."* |
| `clean` | **`fumble`** | **"Oops, dropped it."** (Alternative: `wipe`)<br>Clears cache or temp files.<br>*Output: "Fumbled the cache. It's gone."* |

## 5. Easter Eggs (Hidden Commands)
- **`touch-grass`**: Pauses the terminal for 5 minutes and tells the user to go outside.
  - *Output: "You've been coding for 4 hours. Go touch grass."*
- **`cringe`**: Deletes the `node_modules` folder (dangerous but funny).
  - *Output: "Deleting cringe dependencies... Clean slate."*
- **`fam`**: Lists the team contributors.
  - *Output: "The day ones: [List of names]. Real ones only."*
